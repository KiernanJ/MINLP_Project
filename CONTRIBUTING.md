# Contributing to MINLP Surrogate Project

We welcome contributions to this repository! Here's how you can help:

## 1. Reporting Bugs
If you find a bug, please create an issue. Include:
- A description of the issue.
- Steps to reproduce it.
- Your environment details (Python version, OS, installed packages).

## 2. Suggesting Enhancements
If you have an idea for a new feature or improvement:
- Open an issue describing your suggestion.
- Discuss its potential implementation and value to the project.

## 3. Submitting Pull Requests
If you want to contribute code:
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Write clean, documented code and include tests if applicable.
4. Ensure your code conforms to standard Python style guidelines (PEP 8).
5. Open a Pull Request (PR) and describe your changes in detail.

Thank you for contributing!
